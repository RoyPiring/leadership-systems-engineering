## What this domain covers

Leadership and high-performance frameworks from credentialed coursework, applied as engineered systems. Sources include IPEC, Stanford GSB, PMP, and other formal exec-track development.

**What it isn't.** A replacement for the credential or coursework itself. A coaching service.
